## 回收算法 ##
### 引用计数算法 ###

	资源一旦被引用就进行记录 //例：String name = "张三"; count=0 +1 +1 -1 -1

### 根搜索算法(Roots) ###

- 理解为正在练倒立的一棵大树
- 相比较引用计数算法,这种根搜索算法能够最大限度规避掉内存泄漏


### 补充 ###

- 垃圾回收器,简称GC,主要的工作位置在堆里面,所以堆我们也称之为GC堆
- GC自己本身有固定的回收周期,根据需要程序员可以手动调用垃圾回收器
- 相比较C和Cpp(c++),java在内存管理方面很智能,一般情况下不需要手动进行回收
