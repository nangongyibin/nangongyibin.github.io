## 条件语句(if 翻译过来就是如果，这个判断条件有一个特征 只有两种情况 是的 不是 看见了 没看见) ##

1.第一种句型


#

	if(条件){
		如果条件成立，需要执行的内容
	}
 

2.第二种句型

#

    if(条件){
    	如果条件成立，需要执行的内容
    }else{
    	如果条件不成立，需要执行的内容
    }
 

3.第三种句型

#

    if(条件1){
    	如果条件1成立，需要执行的内容
    }else if(条件2){
    	如果条件2成立，需要执行的内容
    }else{
    	如果条件1和条件2都不成立，需要执行的内容
    }
