- 在继承关系里：父类引用指向子类对象
- 在实现关系里: 接口的实现类可以作为任何参数要求为接口类型的方法的参数
- 基于实现关系的多态在后期的使用场景，我们称之为回调，使用频率很高
- 基于继承关系的多态实际上使用的并不多
- 父类引用指向子类对象 体现在继承关系里 在实现关系里 也是有体现的 同体
