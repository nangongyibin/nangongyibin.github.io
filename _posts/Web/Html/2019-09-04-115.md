## 1、web相关概念 ##

web分类：分动态网站和静态网站，动和静指的是数据
 
做静态网站技术:html css js 

做动态网站技术:asp php javaee(大概包括13门技术 servlet)

web采用架构：bs(browser server)京东 taobao 搜狐，cs(client server) QQ 迅雷
