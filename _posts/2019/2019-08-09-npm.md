### 1、设置npm安装程序时的默认位置 ###

    npm config set prefix "/usr/local/software"


### 2、设置npm安装程序时的缓存位置 ###
    npm config set cache "/usr/local/software/cache"

### 3、在使用npm安装程序时在后面加一个参数-g即可将安装的程序安装到我们指定的目录 ###
