## 数据类型转换 ##

- 当低精度向高精度转换的时候，计算机自动帮助我们进行类型转换(自动转换)
- 当高精度向低精度转换的时候，计算机会提示我们存在异常。
- 当高精度向低精度转换的时候，可以通过强制类型（通过手动添加小括号和类型的方式完成数据类型的转换，结果：损失一部分数据）转换完成。

#

    float----> double 自动
    float------>int 会溢出，小数点后面的会丢失，强制类型转换（数据类型）
