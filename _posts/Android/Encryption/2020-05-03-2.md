### ASCII编码：美国信息标准交互码，就是用来显示西欧字符 ###

![](http://www.nangongyibin.com/assets/images/Encryption/1.png)

#### 获取字符ascii编码 ####

        String str = et.getText().toString().trim();
        char[] chars = str.toCharArray();
        String vs = "";
        for (int i = 0; i < chars.length; i++) {
            char c = chars[i];
            int value = c;
            vs = vs + value + "";
            et.setText(vs);
        }
